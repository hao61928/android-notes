常用快捷键
Ctrl + {或}（左括号或右括号）          
跳到方法或者类的开头或者结尾

Alt + ↑ 或 ↓
在当前.Java文件中的方法间跳转

Ctrl + Alt + ← 或 →
返回或者向前跳到光标所在的位置

F2 或 Shift + F2
显示警告或错误，快速定位错误或者错误

Ctrl + Q
方法、属性、变量、类等的解释，
点击以后，会弹出一个小窗口，里面有这个方法或者属性的一些解释；

Ctrl + Alt + L
格式化当前的代码

Ctrl + Shift +↑ 或 ↓
把当前作用域的代码提前或者后退一
